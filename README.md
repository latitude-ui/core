## installation

#### install packages

To install carbon-button with `npm`:

    npm install carbon-button --save

```jsx 
    import { Button } from "dummy-lib-ibraqr";

    const App = () => <Button label="Click me" />

    export default App;
```

# for Contributors

    npm ci
